  • ArcGIS 10.1 for Server
  • ArcGIS Viewer for Silverlight - Build a Silverlight based map application. This can be done through the app builder in a browser. The builder essentially copies a template application and provides a GUI for editing configuration files, all of which can also be done by hand.
  • ArcGIS Viewer for Flex - Build an Adobe Flex based map application without code. You must log into the server to run the flex builder application to create the web map. The builder essentially copies a template application and provides a GUI for editing configuration files, all of which can also be done by hand.
  • ArcGIS API for Flex - How to write your own custom Adobe Flex based web map and widgets without using the the pre-built viewer. Requires access to Flex development tools from Adobe or Apache.
  • ArcGIS API for Javascript - Build custom web maps that will run in any browser without a plugin (Flex and Silverlight do not work on iPhones and iPads). ArcGIS.com can be used to build a simple a javascript based map without writing any code. That map can then be downloaded and deployed and customized on a local server.
